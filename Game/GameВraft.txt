написание игры:
Стратегия.
Пока что одна общая локация (карта).
2 расы в разных местах карты.
Каждая раса может:
	- иметь одно правительство, которое имеет интерфейс для строительства зданий;
	- строить казармы (интерфейс для подготовки военных юнитов)
		- пехотинцы (1 функция, возможен апгрейд)
		- штурмовики (1 функция, возможен апгрейд)

	- строить школы (интерфейс для обучения гражданских юнитов)
		- скауты (1 функция, апгрейд невозможен)
		- фермеры (1 функция, апгрейд невозможен)
		
	- строить заводы (интерфейс для сборки механизмов)
		- комбайн
		- мотоцикл
		- танк
		
	- строить фермы (интерфейс для выращивания еды)
		- овощи
		- мясо
		- хлеб
		
============================================================================
1. ФАБРИЧНЫЙ МЕТОД.
1.1	Правительство у конкретной расы одно. Поэтому за его создание ответсвтеннен СИНГЛТОН.
		Клиентское приложение - одно и то же.

1.2	С помощью ФМ можно создавать здания. Каждое здание является интерфейсом для создания целевых юнитов
		Клиентское приложение - меню строительства
		Правительство - это "Интерфейс" для создания зданий
		Метод один - построить.
		Строиться либо Казарма, либо Ферма, либо Завод, либо Школа, смотря что указано в параметрах.
		
1.3	С помощью ФМ можно создавать целевых юнитов:
		Клиентское приложение - меню строительства
		"Интерфейсы":
			Казарма
				имеет метод "подготовить"
				реализация 	в классе Пехотинец
							в классе Штурмовик
			Школа
				имеет метод "обучить"
				реализация 	в классе Скаут
							в классе Фермер
			Завод
				имеет метод "собрать"
				реализация 	в классе Комбайн
							в классе Мотоцикл
							в классе Танк
			Ферма
				имеет метод "вырастить"
				реализация 	в классе Овощи
							в классе Мясо
							в классе Хлеб
